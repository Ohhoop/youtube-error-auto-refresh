# YouTube Auto-Refresh on Error

Workaround for YouTube playback errors caused by ad-blocker interactions. When YouTube shows an "An error occurred. Please try again later" message, the extension automatically reloads the page. Capped at 3 reloads per video.

## Install (Opera GX / Chrome / Edge)

**Option 1 - Packed:** download [youtube-error-auto-refresh.crx](https://github.com/Ohhoop/youtube-error-auto-refresh/releases/latest/download/youtube-error-auto-refresh.crx), open `chrome://extensions` (or `opera://extensions`, `edge://extensions`), enable Developer Mode, then drag the `.crx` into the page.

**Option 2 - Unpacked:** clone the repo, open the extensions page, enable Developer Mode, click "Load unpacked" and select the `Unpacked` folder.
